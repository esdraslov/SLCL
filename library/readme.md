# Example library

So you can understand the structure

## MANIFEST

| entry            | value                         | example                                     |
| ---------------- | ----------------------------- | ------------------------------------------- |
| name\*           | mod name                      | testmod                                     |
| description\*    | mod description               | this is a exmaple description               |
| long_description | a longer description          | this is only shown when the mod is selected |
| author\*         | the mod author                | esdraslov                                   |
| created_at\*     | when the mod was created      | 0                                           |
| updated_at\*     | when the mod was last updated | 123123                                      |
| license          | the mod license               | MIT                                         |
| extra            | extra customizeable links     | {"discord":"https[:]//discord[dot]com"}     |
| version          | the API version               | v3 (only supported value)                   |

thumbnail shown is at {modroot}/thumbnail.png (the one used on this example comes from TONE)
